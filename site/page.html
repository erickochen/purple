purple.

SSH config manager and launcher for the terminal

$ curl -fsSL getpurple.sh | sh
or via Homebrew: brew install erickochen/purple/purple
or via cargo: cargo install purple-ssh
Search — fuzzy find across aliases, hostnames and tags
Tags — organize hosts with #tags and filter instantly
Tunnels — manage SSH port forwards per host
Ping — check host reachability from the TUI
Cloud sync — DigitalOcean, Vultr, Linode, Hetzner, UpCloud, Proxmox VE
Round-trip — preserves comments, formatting and unknown directives
Self-update — run purple update, with startup version check
